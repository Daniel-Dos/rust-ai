# Rust Basics

## Rules

- Usar Result<T, E>
- Evitar unwrap()
- Usar async/await

## Style

- Código simples
- Funções pequenas
- estrutura de arquiterara de pasta clara
- deve ser fácil de ler e entender
- deve seguir as convenções de estilo do Rust (rustfmt)
- ler a documentação oficial do Rust para entender as melhores práticas e padrões de código (https://doc.rust-lang.org/book/)
